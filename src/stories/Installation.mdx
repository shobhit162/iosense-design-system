    </div>

    <div className="installation-step">
      <h3>2. Add to Your Project</h3>
      <p>Copy the <code>typography.scss</code> file to your project's styles/tokens directory and import it in your global styles:</p>
      
      <div className="code-block">
        ```scss
        // src/styles.scss
        @use 'styles/tokens/typography';
        ```
      </div>
    </div>

    <div className="installation-step">
      <h3>3. Start Using</h3>
      <p>You can now use the utility classes in your components:</p>
      
      <div className="code-block">
        ```html
        <!-- Example Usage -->
        <h1 class="DisplayLargeRegular">
          Large Display Heading
        </h1>

        <p class="BodyLargeRegular">
          Regular body text with standard spacing
        </p>

        <!-- Combining Classes -->
        <div class="HeadingMediumSemibold global-ls-extended">
          Medium heading with extended letter spacing
        </div>
        ```
      </div>
    </div>

    <div className="usage-examples">
      <h2>Common Usage Examples</h2>

      <div className="example-block">
        <h3>Typography Combinations</h3>
        <div className="code-block">
          ```html
          <!-- Heading with custom spacing -->
          <h1 class="DisplayXLargeRegular global-ps-12">
            Main Heading
          </h1>

          <!-- Body text with letter spacing -->
          <p class="BodyLargeRegular global-ls-normal">
            Regular body text with normal letter spacing
          </p>

          <!-- List with spacing -->
          <ul class="BodyMediumRegular global-lis-16 list-traditional">
            <li>First item</li>
            <li>Second item</li>
          </ul>
          ```
        </div>
      </div>

      <div className="example-block">
        <h3>Responsive Text</h3>
        <div className="code-block">
          ```html
          <!-- Text that adapts to screen size -->
          <h2 class="HeadingLargeRegular">
            This heading will adjust size automatically
          </h2>
          ```
        </div>
      </div>
    </div>

    <div className="troubleshooting">
